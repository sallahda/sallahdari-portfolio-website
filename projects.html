Hephaestus Project

×

Project Overview

Hephaestus is a smart wardrobe assistant developed as a school project where I took responsibility for key components of the infrastructure and backend development. The platform helps users organize and manage their clothing inventory in a streamlined digital environment.

Technical Details

  • Cloud Infrastructure: Designed and implemented the AWS hosting architecture using Terraform as Infrastructure as Code
  • API Development: Created RESTful API endpoints for the Timeline and Profile features
  • CI/CD: Implemented continuous integration and deployment pipeline for automated releases
  • Deployment: Managed the complete deployment process to AWS cloud environment

Key Contributions

  • Architected cloud infrastructure for scalability and performance
  • Wrote Infrastructure as Code using Terraform to ensure consistent deployments
  • Developed backend APIs for user profiles and activity timeline functionality
  • Created CI/CD pipeline for streamlined development workflow

Realtime Election Data Project

×

Project Overview

This data engineering project focuses on processing and analyzing election data in real-time. It leverages Apache Kafka for data streaming, Apache Spark for processing, and PostgreSQL for data storage.

Technical Details

  • Data Streaming: Apache Kafka for real-time data ingestion
  • Processing: Apache Spark for distributed data processing
  • Storage: PostgreSQL for structured data storage
  • Deployment: Docker for local deployment, Terraform for AWS deployment

Key Features

  • Real-time data processing pipeline
  • Fault-tolerant architecture with data replication
  • Scalable design that can handle varying data volumes
  • Flexible deployment options (local or cloud)
